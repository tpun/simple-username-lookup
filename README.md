# Simple Username Lookup Service
This mock service is for in-person interview test. 
NAMES.TXT is from http://www.gutenberg.org/files/3201/files/

## How to run
1. git clone this repository
2. `node app.js`
3. Install [ngrok](https://ngrok.com)
4. `ngrok http 8080`
5. Note the forwarding url (36e14acd.ngrok.io in this case) from the terminal
> Forwarding          http://36e14acd.ngrok.io -> localhost:8080      
>Forwarding           https://36e14acd.ngrok.io -> localhost:8080
